SinglePageApp v1.0
=============

<h1>轻量级单页APP框架</h1>
        
      1.依赖库：zepto.js
      2.构建工具：grunt
      
      build文件夹：存放开发状态（未压缩的JS和图片，未解析的LESS文件）的文件
      css文件夹：存放解析后的css文件
      images文件夹：存放压缩后的图片
      javascript文件夹：存放压缩后的JS文件
      temp文件夹：存放临时文件
      template文件夹：存放模版文件
      app.config.js:配置文件
      
<h1>Lightweight Single Page Web App</h1>

      1.Library dependencies：zepto.js
      2.build：grunt
